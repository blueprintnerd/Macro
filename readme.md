# Macro
### Free plex alternative


# How to use
1. Run the server program and it will guide you through the setup wizard.
2. The setup wizard will automatically create a systemd service to handle the server

3. From then, open the client. 
###### (The default port is 1470)
4. Enter your username, password, and server IP
5. The credentials will save themselves and from then on you can open the client and it will automatically run

### Managing the server
+ After the first time setup, you can configure the app using the json file. 
+ to check the status, you can use the ```mserver status``` command
